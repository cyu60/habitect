# Habitect Universal Index

> Where everything lives across all systems. This is the master cross-reference
> that tells any agent (or human) where to find any type of information.
>
> Part of the **Mnemo** architecture: Input (data ingestion) → View (knowledge interface) → Actuator (take action).
> This file is the **View layer** — the map of all ingested data.

## Universal Life Domains

```
1. VENTURES        — Active projects being built
2. CAREER          — Professional identity, immigration, networking
3. ACADEMIC        — Stanford, JHU, research
4. CREATIVE        — Writing, music, content creation
5. OPERATIONS      — Tools, agents, automation, infrastructure
6. PERSONAL        — Identity, health, relationships, media
7. COMMUNICATION   — Email, messaging (iMessage, WhatsApp, WeChat), calendar
```
