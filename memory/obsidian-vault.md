- Weekly reviews: `- StartDate - EndDate .md`

- Wikilink prefixes use dash not colon: `[[- Jan 2026]]`, `[[W- Date]]`, `[[P- Project]]`, `[[O- Title]]`, `[[A- Article]]`, `[[E- Essay]]`
- macOS does not allow `:` in filenames — all colons in wikilinks replaced with dashes
